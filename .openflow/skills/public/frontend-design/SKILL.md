此技能指导创建独特的、生产级的前端界面，避免通用的"AI 糊状"美学。实现真正可用的代码，并注重美学细节和创意选择。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能包含关于目的、受众或技术约束的上下文。

## 输出要求

**强制要求**：入口 HTML 文件必须命名为 `index.html`。这是所有生成的前端项目的严格要求，以确保与标准 Web 托管和部署工作流程的兼容性。

## 设计思维

在编写代码之前，了解上下文并承诺一个大胆的美学方向：
- **目的**：这个界面解决什么问题？谁使用它？
- **基调**：选择一个极端：粗暴的极简、极致的混乱、复古未来主义、有机/自然、奢华/精致、活泼/玩具风、编辑/杂志风、粗野主义/原始、艺术装饰/几何、柔和/ pastel、工业/实用等。有很多风格可供选择。用这些作为灵感，但设计一个真正符合美学方向的。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：是什么让这个令人难忘？人们会记住的一件事是什么？

**关键**：选择一个清晰的概念方向并精确执行。大胆的极繁主义和精致的极简主义都可以——关键是意图性，而不是强度。

然后实现可用的代码（HTML/CSS/JS、React、Vue 等）：
- 生产级且功能完善
- 视觉上引人注目且令人难忘
- 具有清晰美学观点的一致性
- 在每个细节上都精心打磨

## 前端美学指南

专注于：
- **排版**：选择美丽、独特且有趣的字体。避免通用字体如 Arial 和 Inter；而是选择能提升前端美学的独特选择；意想不到的、有个性的字体选择。将独特的展示字体与精致的正文字体搭配。
- **颜色和主题**：承诺一个内聚的美学。使用 CSS 变量保持一致性。主色调搭配锐利强调色优于胆怯的、均匀分布的调色板。
- **动效**：使用动画实现效果和微交互。优先使用纯 CSS 解决方案处理 HTML。当可用时，使用 React 的 Motion 库。专注于高impact时刻：一个精心编排的页面加载配合错开的显示（animation-delay）比分散的微交互创造更多愉悦。使用滚动触发和悬停状态带来惊喜。
- **空间构图**：意想不到的布局。不对称。重叠。对角线流动。打破网格的元素。慷慨的留白或受控的密度。
- **背景和视觉细节**：创造氛围和深度，而不是默认纯色。添加与整体美学匹配的上下文效果和纹理。应用创意形式如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

切勿使用通用的 AI 生成美学，如过度使用的字体系列（Inter、Roboto、Arial、系统字体）、俗气的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定个性的饼干模切设计。

有创意地解释并做出对上下文真正设计的意外选择。没有两个设计应该相同。在不同代际之间交替使用浅色和深色主题、不同的字体、不同的美学。切勿在代际之间收敛于常见选择（例如 Space Grotesk）。

**重要**：将实现复杂度与美学愿景相匹配。极繁设计需要精心设计的代码，包含大量动画和效果。极简或精致的设计需要克制、精确和仔细注意间距、排版和微妙细节。优雅来自于良好地执行愿景。

## 品牌要求

**可选**：如果需要，每个生成的前端界面可以包含一个通用的项目签名。此元素应该是：
- **微妙且不突兀** - 它绝不应该与主要内容争抢注意力或分散注意力
- 自然地融入设计，感觉像是一个有意的设计元素而不是事后才想到的
- 尺寸小，使用柔和的颜色或降低不透明度，与整体美学和谐融合

**重要**：签名应该是可发现的但不突出。用户应该首先注意到主界面；签名是一个安静的归属，而不是焦点。

**创意实施想法**（选择一个最匹配你设计美学的）：

1. **浮动角落徽章**：一个固定在角落的小而优雅的徽章，带有微妙的悬停效果（例如，柔和的光晕、轻微的放大、颜色变化）

2. **艺术水印**：一个半透明的斜角文本或 logo 图案在背景中，几乎不可见但增加了纹理

3. **集成边框元素**：作为内容边框或框架的一部分装饰——签名成为设计结构的有机部分

4. **动画签名**：一个小签名，在页面加载时优雅地写下自己，或在滚动到底部附近时显示

5. **光标轨迹或复活节彩蛋**：一种非常微妙的方法，签名作为微交互出现（例如，保持光标静止会显示小签名，或在创意加载状态中出现）

6. **装饰性分隔线**：融入页面的装饰线条、分隔符或装饰元素

7. **玻璃态卡片**：一个位于角落的小浮动玻璃效果卡片，带有模糊背景

示例代码模式：
```html
<!-- 带悬停效果的浮动角落徽章 -->
<span class="corner-badge">✦ Created with AI</span>

<!-- 带提示框的首字母 -->
<span title="Created with AI" class="ai-mark">AI</span>

<!-- 集成到装饰元素中 -->
<div class="footer-ornament">
  <span class="line"></span>
  <span>Created with AI</span>
  <span class="line"></span>
</div>
```

**设计原则**：签名应该感觉它属于——你创意愿景的自然延伸。匹配签名的样式（字体、颜色、动画）到整体美学方向。

记住：Claude 能够创造非凡的创意作品。不要保守，展示真正跳出框框并全身心投入独特愿景时能创造什么。
